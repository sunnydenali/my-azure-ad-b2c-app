my-azure-ad-b2c-app

This repository uses esbuild to produce a production bundle in the dist/ folder.

  1. Build: npm run build
  2. Serve: npm start (serves the contents of dist/)

After building, open http://localhost:8080 (the server serves the built app).

If you want a quick preview without running the server, open dist/index.html in a browser (recommended to run the server for correct routing).
